# UI/UX Pro Max Skill (v2.0)

This skill provides an AI-powered reasoning engine for generating tailored design systems and high-quality UI implementations.

## Capabilities

### 1. Design System Generation
Analyzes project requirements to generate:
- **Patterns**: Landing page structures, layouts, conversion elements.
- **Styles**: 67+ UI styles (Glassmorphism, Bento Grid, Soft UI, etc.).
- **Colors**: 96+ Industry-specific palettes.
- **Typography**: 57+ Font pairings with Google Fonts.
- **Effects**: Animation curves, hover states, shadows.

### 2. Industry-Specific Reasoning
Applying 100+ specialized rules for categories including:
- **Tech & SaaS**: B2B, Developer Tools, AI Platforms.
- **Finance**: Fintech, Banking, Crypto.
- **Healthcare**: Clinics, Mental Health.
- **E-commerce**: Luxury, Marketplace.
- **Services**: Spa, Legal, Consulting.
- **Creative**: Portfolios, Agencies.
